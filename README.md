# Customer Retention & Churn Analytics

## Business Problem

A telecommunications company is experiencing customer churn.

Leadership wants to understand:

- Which customers are leaving?
- Why are they leaving?
- Which customer segments are most valuable?
- What actions could improve retention and customer lifetime value?

## Project Objective

Analyze customer behavior, churn patterns, and customer lifetime value to identify opportunities for improving retention and revenue.

## Tools

- SQL (PostgreSQL)
- Python (Pandas)
- Tableau

## Planned Analyses

1. Customer Overview
2. Churn Analysis
3. Revenue Analysis
4. CLTV Analysis
5. Geographic Analysis
6. Customer Segmentation
7. Business Recommendations

IBM Telco Customer Churn Dataset
